# QC Hub

Maya 用 QC ツール群の統一ランチャー。Model QC Tools・UV QC Tools・Scene Cleanup Tools を1つのウィンドウからワンクリックで起動できる。

## 基本の流れ

1. QC Hub を起動
2. 使いたいツールの **ボタンをクリック**
3. 対象ツールのウィンドウが開く

---

## 機能

- **ツール起動**: 各ボタンをクリックすると対応するツールの `launch()` が呼び出される
- **全て閉じる**: 起動中の全ツールウィンドウ（結果ウィンドウ・ヘルプダイアログ含む）を一括で閉じる
- **エラー通知**: モジュールが見つからない場合はエラーメッセージを表示

---

## 対象ツール

| ツール | 説明 |
|---|---|
| Model QC Tools | モデル品質チェック |
| UV QC Tools | UV 品質チェック |
| Scene Cleanup Tools | シーン整理チェック |

新しいツールの追加はボタン定義リストに1エントリ追加するだけで対応可能。
