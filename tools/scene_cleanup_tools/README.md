# Scene Cleanup Tools

Maya 用のシーン整理ツール。ジオメトリ・不要ノード・シーン環境を一括チェックし、検出結果の確認・選択をインタラクティブに行える。

## 基本の流れ

1. シーンを開いた状態でツールを起動
2. チェック項目を確認（個別に ON/OFF 切り替え可能）
3. **チェック** ボタンを実行
4. 結果ウィンドウで検出された問題を確認・選択
5. すべてのチェックは非同期で実行され、プログレスバーで進捗を確認できる。処理中は×ボタンでキャンセル可能。

---

## チェック範囲

シーン内の全メッシュノードが自動的にチェック対象になる。

> ℹ️ シーン全体項目（空グループ、未使用マテリアル、未使用レイヤー、空セット、ネームスペース、不明ノード、リファレンスノード、重複名、シーン単位）は常にシーン全体をスキャンする。

---

## チェック項目（17）

各項目は個別に ON/OFF 切り替え可能。**すべて ON** / **すべて OFF** / **デフォルトに戻す** で一括操作。

### ジオメトリ（7）

ヒストリ、トランスフォーム、頂点 Tweaks、インスタンス、スムースプレビュー、Shape Suffix、重複名。

### 未使用・不要ノード（6）

空グループ/空シェイプ、中間オブジェクト、未使用マテリアル、未使用レイヤー、空セット、ネームスペース。

### シーン環境（4）

シーン単位、不明ノード、リファレンスノード、命名チェック。

---

## 結果ウィンドウ

- 項目クリック: 該当箇所がビューポートで選択される
- エラーなし → 「エラーなし ✔」ダイアログを表示
- ステータスバーにチェック名と処理時間を表示
