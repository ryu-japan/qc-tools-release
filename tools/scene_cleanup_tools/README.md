# Scene Cleanup Tools

Maya 用のシーン整理ツール。ジオメトリ・不要ノード・シーン環境を一括チェックし、検出結果の確認・修正をインタラクティブに行える。

## 基本の流れ

1. シーンを開いた状態でツールを起動
2. チェック項目を確認（個別に ON/OFF 切り替え可能）
3. **チェック** ボタンを実行
4. 結果ウィンドウで検出された問題を確認
5. 必要に応じて **選択項目を修正** で修正

チェックは非同期実行。プログレスバーで進捗を確認でき、×ボタンでキャンセル可能。

---

## チェック項目

### ジオメトリ

残存ヒストリ、未フリーズトランスフォーム、頂点Tweaks残留、インスタンス残存、スムースメッシュプレビュー

### 未使用・不要ノード

中間オブジェクト、空グループ/空シェイプ、未使用マテリアル/テクスチャ、未使用レイヤー、空オブジェクトセット、空ネームスペース

### シーン環境

不明ノード、リファレンスノード残存、シーン単位/Up軸、ファイルパスチェック

---

## 結果ウィンドウ

- 左パネルでチェック項目を切り替え、右パネルで検出ノードを確認
- ノードクリックで Maya 上のオブジェクトを選択
- リスクのあるノードは ⚠ アイコンと色で警告表示
- **選択項目を修正** で選択したノードを一括修正（リスクに応じた確認あり）
