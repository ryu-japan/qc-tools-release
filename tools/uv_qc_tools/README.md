# UV QC Tools

Maya 用の UV 品質チェックツール。複数のチェック機能を備え、検出結果の確認・修正をインタラクティブに行える。

## 基本の流れ

1. チェック対象の **メッシュを選択**（トップノード選択で子メッシュを自動収集）
2. 必要に応じて **準備** を実行（UVSet確認・1メッシュ1マテリアル化）
3. **テクスチャ解像度** を設定（自動検出も可能）
4. 各 **チェック** ボタンを実行
5. 結果ウィンドウでエラーを確認・選択
6. すべてのチェックは非同期で実行され、プログレスバーで進捗を確認できる。処理中は×ボタンでキャンセル可能。

---

## 準備（Preparation）

### UVSet チェック

- 選択メッシュの全UVSet（map1含む）を一覧表示
- 各UVSetの該当メッシュ数を表示。デフォルト(map1)にはマーカー付き
- クリックでメッシュを選択し、UVエディタのUVSetを切替
- 削除機能なし（誤削除防止のため手動で対応）

### 1メッシュ1マテリアル化

- メッシュが複数マテリアルを含む場合、マテリアルごとに分割してからチェックすると確認しやすくなる
- **1メッシュ1マテリアル化** ボタンでワンクリック実行。**⚙ ボタン** で詳細設定
- 完了時にマテリアル名・メッシュ数・グループ名のサマリーを表示

### テクスチャサイズ

- プリセットから選択、またはCustomで任意の数値を直接指定
- **テクスチャから自動検出**: 選択メッシュのテクスチャから解像度を自動取得
- **非正方形**: チェックを入れると X / Y を個別に指定可能

---

## チェック（Check）

### ピクセルエッジ整列

UV境界エッジがピクセル境界線に揃っているかチェックする。

- 水平(H) / 垂直(V)の境界エッジ（シーム含む）が対象
- 連続エッジを自動グループ化して1項目として表示
- **最小エッジ長(px)**: この長さ未満のエッジは除外（デフォルト: 1.0）
- 結果ウィンドウでチェックボックスで個別選択・解除が可能
- **選択をピクセル境界にスナップ**: チェックしたエラーをピクセル境界に自動修正（Ctrl+Zで元に戻せます）

### UVパディング

UVシェル間およびUDIMタイル境界からのパディング距離をチェックする。

- **無視する距離 (px)**: この距離未満は無視（デフォルト: 0.05、ロック状態）。「手動調整」で解除
- **Shell padding — エラー距離 (px)**: シェル間でこの距離未満をエラー検出（デフォルト: 5）※整数のみ
- **Tile padding — エラー距離 (px)**: タイル境界からこの距離未満をエラー検出（デフォルト: 2）※整数のみ
- 結果クリック → エラー箇所の頂点がハイライト
- **通常モード**（デフォルト）: 高速かつ高精度。ほとんどの場合はこちらで十分
- **高精度モード**: フル解像度でチェック。通常モードと結果が異なる場合に使用

### UVレンジチェック

UVフェースがUDIMタイル境界をまたいでいないか、有効範囲外にないかをチェックする。

- **タイルまたぎ検出**: フェースが複数タイルにまたがる場合にエラー
- **有効範囲外検出**: フェースが有効UV範囲外に完全に存在する場合にエラー
- UV U / V: min ~ max をそれぞれ指定（デフォルト: 0 ~ 1）。マイナス値対応
- 両方のチェックを個別に ON/OFF 可能

### UV重複

UVフェースの重複を検出する。

- 結果は「⚠ 要確認の重複」と「ℹ 意図的と推定」の2グループに自動分類して表示。クリックで該当シェルの全フェースを選択
- **同一シェル内の重複も含める**: ONで同一シェル内の自己重複も検出（デフォルト: OFF）

### テクセル密度

テクセル密度 (px/unit) を計測する。モードを選んで計測すると、結果ウィンドウに統計とヒストグラムが表示される。

- **通常モード（シェル単位）**: シェルごとの面積加重平均で密度を計測。仕様上のテクセル密度に対する適合チェックに使用（デフォルト）
- **均一性モード（フェース単位）**: フェースごとに密度を計測。シェル内の極端な密度ムラを検出する補助チェック
- **結果はフィルタ表示**: エラーリストにはフィルタ条件に該当するものだけが表示される。条件を変えるとリストも即座に更新される
- デフォルトでは平均密度の20%未満の小さいシェルを検出（%は調整可能）
- **基準値 ± 許容値で設定**: 目標密度からのズレをチェックしたい場合に
- **下限・上限を個別指定**: 任意の範囲でチェックしたい場合に。ヒストグラム上のドラッグでも調整可能
- エラーはヒートマップ色（低=赤、高=青）で色分け表示。クリックでビューポート選択

> ⚠️ Mayaの浮動小数点の特性上、テクセル密度の値にはごく僅かな揺らぎがあります。フィルタ境界ぎりぎりの値が意図せず検出される場合は、フィルタ範囲を少し広めに設定してください。

### UV方向チェック (Beta)

各UVシェルのワールド空間上の上下とUV空間上の上下の対応を検証し、回転ずれや反転を検出する。

- メッシュごとに面積最大の主要面を基準に上下方向を決定
- 主要面が垂直に近い場合は面積加重法線から上下を決定、水平に近い場合は手前がUVの下、奥がUVの上に相当
- **正常 (Normal)**: UVの上方向がメッシュの上方向と±45°以内で一致
- **回転ずれ (Rotated)**: UVの上方向が±45°を超えてずれている（上下逆転・90°回転等）
- **(反転)**: UVが反転しているシェルはリストに「(反転)」ラベルが付く
- **判定不能 (Indeterminate)**: 退化フェースなどで判定不能
- 各エントリにメッシュ名・シェルIDを表示。クリックで該当フェースを選択

## 結果ウィンドウ

- 項目クリック: 該当箇所がビューポート / UVエディタで選択される
- **すべて選択**: 全エラーを一括選択
- エラーなし → 「エラーなし ✔」ダイアログを表示
- ステータスバーにチェック名と処理時間を表示
- テクセル密度は専用の結果ウィンドウで、ヒストグラムとフィルタで条件を調整できる

---

## レポートを送信

実行済みチェックの結果と環境情報をフィードバックフォームに送信する。

- ブラウザでフォームが開き、レポートが自動入力される
- レポートが長すぎる場合はクリップボードにコピーされるので、フォームのレポート欄に貼り付けてください
