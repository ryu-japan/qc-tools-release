# QC Tools Release

Maya 向け QC ツール群の配布用リポジトリ。

## 構成

- `manifest.json` — 各ツールの最新バージョン・ダウンロード URL・SHA-256 ハッシュ
- `tools/<tool_name>/` — 各ツールの成果物（.py）と README

## 対象ツール

| ツール | 説明 |
|---|---|
| qc_hub | QC ランチャー |
| qc_tools_plugin | Maya メニュープラグイン |
| model_qc_tools | モデルチェックツール |
| uv_qc_tools | UV チェックツール |
| scene_cleanup_tools | シーン整理ツール |
